NLFB - Nice Localhost File Browser

NLFB is a simple one-page web "app" that gives you a pretty nice interface to browse you projects

It comes comes with different useful features, such as

  • A side bar, listing all your projects listed
  • A special search bar
  • Breadcrumbs and type path "feature"
  • A colored code previewer
  • Other things...

Ignoring items

You can ignore different items:

  • projects: affect the side bar, cannot be show
  • folders: affect the main panel. You can show them by right clicking and choosing Toggle Hidden Items
  • files: same as the folders

You can add/remove ignored items in the config.json file at the root of the project. It should look something like this:

{
	"exclude_projects": [
		"_nlfb"
	],
	"exclude_files": [
		".git"
	],
	"exclude_folders": []
}
