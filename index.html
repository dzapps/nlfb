NLFB - Nice Localhost File Browser

NLFB is a simple one-page "app" that replace this just all right localhost homepage but also this disgusting Apache made browser.

This browser comes with different useful features, such as

  • A side bar, listing all your projects listed
  • A special search bar
  • Breadcrumbs
  • A colored code previewer

The special search bar

It search for folders and files through the current directory. The special point is that it is based on the order of the letters, so says you type ctrl, it would bring up controller (if there's any of course)
